### **如果需要降級到 Tailwind CSS v3**:
```bash
# 移除 Tailwind CSS v4
npm uninstall tailwindcss @tailwindcss/postcss

# 安裝 Tailwind CSS v3
npm install -D tailwindcss@3 postcss autoprefixer

# 更新配置文件
npx tailwindcss init -p

# 修改 globals.css
# 移除 @import "tailwindcss";
# 添加傳統的 Tailwind directives
```
