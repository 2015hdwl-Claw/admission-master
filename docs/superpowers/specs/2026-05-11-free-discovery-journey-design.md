# 免費發現之旅 v2.0 設計規格

## 核心價值主張

### 與其他升學網站的根本差異

**其他所有升學網站：**
> 根據你現在的成績，你可以上這些學校。接受現實吧。

**升學大師：**
> 不管你現在在哪裡，你想要去的那個科系，有 6 種不同的路可以抵達。
> 
> 我們不會告訴你你辦不到。我們只會告訴你每一條路需要付出什麼代價。然後你自己選擇。

---

## 介面設計原則（完全遵循光域風格）

### 原則 1：一次一個問題
- 全螢幕，沒有導覽列，沒有其他連結
- 一次只顯示一個輸入項，使用者不會分心
- 90% 的畫面是內容，10% 是介面

### 原則 2：沒有按鈕的介面
- 完成輸入後自動前進
- 滑動就可以到下一步/上一步
- 永遠不會看到「送出」、「下一頁」這種按鈕

### 原則 3：流暢的狀態過渡
- 每個步驟進入時有淡入+滑入動畫
- 進度條頂部平滑增長
- 背景顏色隨步驟細微變化，創造前進感

### 原則 4：隨時可以回頭
- 任何時候都可以滑回去修改之前的答案
- 修改後所有下游計算自動更新
- 沒有「不能返回」的步驟

### 原則 5：沒有評斷
- 永遠只說「如果你想要這個，你需要做什麼」
- 不會出現「你目前的條件不夠」這種訊息
- 差距 = 機會，不是失敗

## 六步驟發現流程

### 步驟 1：我想去哪裡？
> 瀏覽大學科系，選擇 3 個有興趣的目標

**介面**：
- 全螢幕單一搜尋框，字體大而溫暖
- 輸入即時出現科系卡片，點擊直接選取
- 選取 1-3 個後自動前進到步驟 2
- 每個科系顯示：「這個科系有 X 種入學管道」

**光域風格**：
- 背景色：淡藍漸層
- 無導覽列，全螢幕沉浸
- 搜尋結果以卡片浮現，帶有淡入動畫

### 步驟 2：這個目標需要什麼？
> 系統拆解該科系的所有入學管道和門檻

**介面**：
- 橫向滑動的 6 張管道卡片
- 每張卡片：管道名稱、圖標、核心門檻、去年錄取率、截止日期
- 點擊任一管道展開詳細資訊（申請時程、文件清單、分數分布）
- 完全匿名，不需要註冊或登入

**6 種管道**：

| 管道 | 核心條件 | 截止日期 | 去年錄取率 |
|------|---------|---------|-----------|
| 📚 繁星推薦 | 在校成績前 30% | 10/15 | 18% |
| 🎯 甄選入學 | 統測 PR 55+ | 12/20 | 12% |
| 📝 聯合登記 | 統測成績 | 04/10 | 8% |
| 🏆 技優甄審 | 乙級證照 | 03/05 | 42% |
| ⭐ 技優保送 | 競賽前三名 | 02/20 | 91% |
| 🎓 特殊選才 | 特殊經歷 | 01/15 | 27% |

**光域風格**：
- 背景色：淡紫漸層
- 6 張卡片垂直排列，滑動瀏覽
- 每張卡片有獨特的漸層色彩

### 步驟 3：我現在有什麼？
> 盤點現有的積分、證照、成績

**介面**：
- 一次只出現一個問題，全螢幕顯示
- 選擇後自動前進到下一題
- 問題清單：
  1. 你的年級？
  2. 在校成績大約在多少百分比？
  3. 你有任何技術士證照嗎？
  4. 你有參加過任何技藝競賽嗎？
  5. 你有做過專題或研究嗎？

**光域風格**：
- 背景色：淡綠漸層
- 每題之間有滑動過渡動畫
- 頂部進度條隨答題數平滑增長
- 選項以大面積可點擊區域呈現

### 步驟 4：我的差距在哪裡？ ✨ 魔法時刻 ✨
> 這是整個系統最重要的畫面

**介面**：
```
🎯 目標：台科大電子系 技優甄審

✅ 你已經具備的：
   ✓ 年級符合資格
   ✓ 在校成績達到門檻

⚠️ 你需要加強的：
   🟡 乙級證照 - 下次考試在 3/15，還有 72 天可以準備
   🟡 專題作品 - 需要加強實作內容
   🔴 讀書計畫 - 完全還沒有準備

📊 目前錄取機率：38%

💡 如果你完成這三件事：
   👉 預估錄取機率：76%
```

**光域風格**：
- 背景色：淡金漸層
- ✅ 項目以綠色柔和呈現
- ⚠️ 項目以黃/紅溫和提醒
- 機率數字以大字體動畫呈現
- 永遠在最後強調「如果你完成這些，你可以到達」

### 步驟 5：我需要完成什麼？
> 產生個人化的待辦清單

**介面**：
- 按優先順序排列的待辦事項
- 每個項目有截止日期和剩餘天數
- 勾選完成有滿足感動畫
- 分為：🔴 重要 / 🟡 中等 / 🟢 一般

**光域風格**：
- 背景色：淡橙漸層
- 待辦項目以卡片形式浮現
- 完成勾選帶有粒子效果

### 步驟 6：我的時間表是？
> 倒數計時的行動計畫

**介面**：
```
🚀 接下來三個月你需要做的事：

▢ 02/15 之前：完成乙級證照報名（還有 45 天）
▢ 02/28 之前：完成專題報告初稿（還有 58 天）
▢ 03/10 之前：考乙級證照（還有 68 天）
▢ 03/20 之前：完成讀書計畫（還有 78 天）
▢ 03/30 之前：送出申請（還有 88 天）

💾 儲存我的計畫  |  📤 分享給導師  |  ⏰ 設定提醒
```

**光域風格**：
- 背景色：淡粉漸層
- 時間線以垂直軸呈現
- 每個里程碑帶有倒數計時
- 底部三個行動按鈕

## 技術架構

### 前端架構
- 單頁 React 元件，完全無頁面跳轉
- 全螢幕步驟介面，一次只顯示一個問題
- 流暢的滑動過渡動畫
- 背景顏色隨步驟漸變，營造旅程感

### 狀態管理
- 所有狀態儲存於 localStorage
- 隨時可以離開、回來、繼續
- 不需要後端連線即可完成完整流程
- 登入後可以同步到伺服器

### 推薦演算法
1. **絕對門檻過濾器**：先移除完全不符合資格的管道
2. **適合度評分器**：計算每個管道的相對優勢
3. **差距分析器**：精確計算每個條件的缺口和可行性
4. **機率預測器**：基於歷史數據的錄取機率估計
