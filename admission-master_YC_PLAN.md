# 升學大師 v2.0 - YC 5-Phase 主計畫書 (功能融合策略版)

**專案名稱**: 升學大師 v2.0 (Admission Master)  
**計畫期間**: 2026-05-11 ~ 2026-06-07 (4週) - **簡化版**  
**產品定位**: 高職升學路徑發現引擎  
**目標市場**: 台灣高職生 (免費) + 家長 (付費)  
**開發策略**: **功能融合並用** - 重新定位舊功能，加速開發

---

## Executive Summary

升學大師 v2.0 是一個**高職升學路徑發現引擎**，解決 99% 高職生不知道自己有 6 種升學管道的資訊不對稱問題。

**核心策略變革**: 從「全新開發」轉變為「功能融合並用」策略，由於舊功能沒有用戶，我們可以直接重新定位並融合現有頁面，大幅縮短開發時間。

**時間優化**: 從原本的 6 週縮短為 3-4 週，快速部署到生產環境。

**核心價值**: 3 分鐘免費測試，發現你的潛在升學路徑，不再遺憾。

## Phase 1: 構思與確認 (Idea Validation)

### 1.1 重新定位決策

**重新定位**: 從「學習歷程管理平台」→「升學路徑發現引擎」

**關鍵洞察**: 
- 舊功能沒有用戶，可以自由重新定位
- 現有技術架構穩定，無需從頭開發
- 許多功能可以重新定位為 v2.0 服務

### 1.2 功能融合策略

**核心決策**: **融合並用** - 重新定位 8 個有潛力的舊頁面，移除 8 個無用頁面

**保留並重新定位的頁面**:
- `/explore` → **職群探索** (Explorer)
- `/portfolio` → **路徑準備材料** (Planner)
- `/ability-account` → **Planner 能力中心** (Planner)
- `/roadmap` → **升學路徑時間線** (Planner)
- `/interview` → **個人申請準備** (特定路徑)
- `/pathways` → **6 種管道總覽** (教育)
- `/department/[slug]` → **職群科系介紹** (內容)
- `/quiz` → **職群發現測驗** (Explorer)

**移除的頁面**:
- `/analyze`, `/business-strategy`, `/calendar`, `/guide`, `/results`, `/strategy`, `/teacher`, `/test`, `/demo`

**暫時隱藏的頁面**:
- `/parent`, `/parent/register`, `/pricing` (Phase 3 再啟用)
