## 🔍 已嘗試方案記錄

### 方案 1: 全局 CSS 強制規則 ❌ 失敗
**Commit**: fca197b (2026-05-02 23:15)  
**實施內容**: 在 globals.css 添加 !important 強制規則

```css
*, *::before, *::after {
  writing-mode: horizontal-tb !important;
  text-orientation: mixed !important;
  direction: ltr !important;
}
```

**結果**: 無效，問題持續存在

### 方案 2: 組件級 Inline Styles ❌ 失敗
**Commit**: 7ea0b78 (2026-05-02 23:39)  
**實施內容**: 在 login/page.tsx 添加 inline styles

```tsx
<div style={{ writingMode: 'horizontal-tb', textOrientation: 'mixed', direction: 'ltr' }}>
```
